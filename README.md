# phase-3-shoppe
Phase 3 Rails Online Shop App
User Stories:
1. As a user I want:
  - Register
  -Login/Logout
  -Go to my user account
    - Edit my user account
  - View products
  - * Search for products
  - Place item in a shopping cart
  - Purchase items in shopping cart
  - * See past purchases
2. As an admin I want:
  - To login/logout
  - Post items
  - Edit items
  - Remove items
